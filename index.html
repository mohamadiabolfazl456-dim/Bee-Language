About Bee

Bee is an experimental programming language created for learning, research, and exploring how programming languages work internally. It includes its own lexer, parser, runtime, and interpreter, all written from scratch in modern C++.

Features

⚡ Fast Interpreter

Custom lexer, parser and interpreter.

📦 Built-in Types

int, float, bool, string, list and struct.

🔧 Utilities

Math, string, list, file and console functions.

🌍 Cross Platform

Runs on Windows, Linux and Android.

Project Goals

Bee is designed to understand every stage of programming language implementation—from lexical analysis and parsing to runtime execution. Future plans include:

  • Classes
  • Modules
  • Package Manager
  • SDL2 Graphics
  • Bytecode Virtual Machine
  • Garbage Collection
